Using Adobe Photoshop

In class we worked on making different layers in Adobe Photoshop. Our project was to put our face on top of an image and making it as realistic as possible. Using the refine edge tool I was able to cut my head out of an original image of me and transfer it to the image of Lionel Messi. When my head was layered over Messi's face the neck seemed paler so i decided to use the brush tool and make it darker to match my skin color.
